# Captura-de-Leads

Ferramenta de caixa de entrada para triagem e gestão de leads do Juris8 DET Monitor.

- Conectada ao Supabase
- Pipeline de status (Novo → Em contato → Qualificado → Ganho / Perdido)
- Anotações internas
- Ações rápidas (WhatsApp + E-mail)
- Busca e filtros

## Deploy

Deploy no Vercel após o push:

```bash
echo "# Captura-de-Leads" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/eucaiogomes/Captura-de-Leads.git
git push -u origin main
```

Depois vincule o repositório no Vercel.
"# Captura-de-Leads"
